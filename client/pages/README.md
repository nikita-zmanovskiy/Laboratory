Pages

Слой `pages` содержит страницы приложения, собранные из `widgets`, `features`, `entities` и `shared`

В этом проекте используется App Router из Next.js, поэтому реальные маршруты находятся в директории `client/app`
Директория `src/pages` не отвечает за routing напрямую. Здесь хранятся page-композиции, которые подключаются в файлах `app/**/page.tsx`

Назначение слоя

`pages` отвечает за:

- сборку страницы из готовых блоков
- подключение крупных виджетов
- размещение layout-разметки страницы
- передачу данных и обработчиков между виджетами
- минимальную page-level логику

Что можно хранить в `pages`

Внутри страницы можно держать:

- основной JSX страницы
- композицию виджетов
- page-specific hooks в `model`
- page-specific helpers в `lib`
- CSS-модули, относящиеся только к этой странице
