# Папка pages

Эта папка необходима для корректной работы Next.js App Router.

Даже если вы используете App Router (папка `app`), Next.js будет пытаться использовать `src/pages` в качестве Pages Router, если эта папка отсутствует в корне проекта, что приведёт к ошибкам при сборке проекта.

Согласно документации FSD для Next.js, эта папка должна существовать в корне проекта, даже если она пустая.
